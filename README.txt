
1 - Install NPM and NodeJS if not installed

2 - Execute 'install.bat' with administrator privileges.

3 - Execute 'run_clr.bat' to run the application.

4 - The CLR Endpoint should be running on localhost, port 8197.

5 - If the browser didn't open when executed 'run_clr.bat', open 'http://localhost:8197/' to select and manage available offers!
